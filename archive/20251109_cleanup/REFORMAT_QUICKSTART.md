# 🚀 Web界面"重新排版"功能 - 快速上手

## 5分钟快速开始

### 1. 启动服务 (如果未运行)

```bash
# 确保 Ollama 服务运行中
curl http://localhost:11434/api/tags

# 启动 Web 服务
python scripts/run_web.py
```

### 2. 访问文章

打开浏览器访问任意文章：
```
http://localhost:7888/article/1
```

### 3. 点击"重新排版"

找到文章内容上方的按钮：
```
[编辑文章] [重新排版] ← 点击这个
```

### 4. 等待处理 (10-20秒)

看到状态提示：
```
正在排版... ⏳
```

### 5. 查看结果

排版完成后：
- ✅ 显示成功消息
- 📝 内容自动更新
- 💾 结果自动保存
- ↩️ 可点击"撤销排版"恢复

## ✨ 特点

- **轻量级** - 只修正格式，不改内容
- **快速** - 10-20秒完成
- **安全** - 可随时撤销
- **本地** - 使用本地LLM，无隐私风险
