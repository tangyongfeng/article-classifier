那么， 特 斯 拉 公司 的 高明 之处 到底 在哪里 呢？ 关键 的 技术 就在 这个 电池 组 的 管理 上。 对于 电池 组 的 管理 而言， 每个 电池 很难 做到 电压 完全 一致， 当 电池 电压 不一致 的 时候， 连接 在一起 的 电池 会 产生 压差， 而 高 电流 会 通过 那些 电压 最小 的 电池， 从而 产生 电池 发热， 甚至 爆炸。 特 斯 拉 的 高明 之处 在于 它 使用 了 大量 传感器 和 分析 软件， 可以 实时 监测 每一个 电池 组 的 电压， 并且 通过 软件 分析 技术 和 控制 技术， 自动 调节 电流， 从而 使得 电压 和 电流 在 电池 组 之间 保持 均衡。 因此， 与其说 特 斯 拉 发明 的 电动 汽车 采 用的 是 顶尖 的 汽车 技术， 倒不如 说 它 依赖 的 是 先进 传感器 和 软件技术。 从根本上 说， 特 斯 拉 的 汽车 就是 一个 基于 软件 和 传感器 的 移动 互 联 终端。
