使用逗号间隔

后代（子项）选择器，
使用空格间隔
，这种方式可以跳过多重间隔生效，例如body中包含em ，则可以使用body em  { color:read}来定义包裹再body中的em 的样式。
父子选择器，
使用>间隔
，这种方式只能在标签的父子关系时生效。例如 p>strong
相邻选择器，
使用+间隔
，这种方式只能在两个相邻的标签中发挥作用。例如 p+h1
兄弟选择器，
使用~间隔
，这种方式将会在间隔之后的每一个相同的标签发挥作用，例如h1~h2 ，实现的效果则是h1h2h2h2h2h2h2h2h2h2h2 只要相邻的相同标签，都会有效果。
属性选择器，html中， <h1 title="demo">什么<h1>中的demo 就是这个标签的属性，可以直接通过属性的分类来定义样式.
使用[]标记
 。例如：h1[title]{color:red;} 。进阶，还可以根据属性的值来设定样式，使用方法如: hi[tile="demo"] {color:red;}

css长度单位
写成div.size表示应用了size类的div。
像素px ：就是像素了
em,指定文字是父级或者默认大小的倍数。1em 一倍。 2em两倍
rem 相对根元素（html）的大小
vw 是相对与窗口宽度的百分单位.和高度无关，只和宽度有关。
vh是相对与窗口高度的百分单位
vmax 是指文字的大小宽度和高度相比
较大者
有关。调整浏览器的宽度或者高度，改文字的大小也会随着浏览器的大小而变化。
vmin是指问的大小宽度和高度相比较小者有关，调整浏览器的宽度或者高度，文字的大小也会随着浏览器的大小变化。
%之前的标准定义%是一个长度单位，但是现在标准已经被修订为%是一个数据类型。100%=1em

html结构。section article 是html5增加的标签，是用于在页面中体现页面中所重点所需要说明的内容
div、section、article 之间的区别：div、section、article：语义从无到有，逐渐增强
div：无任何语义，仅用作样式化和脚本化的标签。
section：主要用于一段主题性内，一般包括一个或多个标题。
article： 主要描述一段具体的、可以脱离上下文的内容。
nav，aside区别
nav元素是一个可以用作页面导航的链接组，其中的导航元素链接到其他页面或当前页面的其他部分。并不是所有的链接组都要被放进nav元素，只需要将主要的、基本的链接组放进nav元素即可。

定位
position
absolute  生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。 元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
fixed 生成固定定位的元素，相对于浏览器窗口进行定位。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
relative 生成相对定位的元素，相对于其正常位置进行定位。因此，"left:20" 会向元素的 LEFT 位置添加 20 像素。
static 默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
sticky 粘性定位，该定位基于用户滚动的位置。它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。
          注意: Internet Explorer, Edge 15 及更早 IE 版本不支持 sticky 定位。 Safari 需要使用 -webkit- prefix (查看以下实例)。
inherit 规定应该从父元素继承 position 属性的值。
initial 设置该属性为默认值，详情查看 CSS initial 关键字。

left

right

none
默认值。元素不浮动，并会显示在其在文本中出现的位置。
inherit
规定应该从父元素继承 float 属性的值。

将指定区域按照网格形式展示，
命令
display:grid   声明开始网格显示
grid-gap 网格的间距
grid-column-gap 列间距
grid-row-gap 行间距
grid-template-rows 200px 200px 200px  # 分别表示网格行的长宽高
grid-template-columns 200px 200px 200px  #网格行的长宽高
可用等分单位 fr 例如grid-template-rows 1fr 1fr 1fr  ，表示在父容器给与的空间中等分1份；
亦可 grid-template-row repeat(3,1fr) 表示写三遍
亦可 grid-template repeat(3,1fr)/ repeat(3,1fr)        #两个repeat之间间隔使用/完成。

如果出现该属性，则当媒介文件完成播放后再次开始播放。

如果出现该属性，视频的音频输出为静音。

auto
metadata
none
