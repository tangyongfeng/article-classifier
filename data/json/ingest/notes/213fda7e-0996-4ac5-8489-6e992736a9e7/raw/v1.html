绑定类型
     outlet类型用于绑定控件属性，把ui控件的属性和代码绑定在一起，方便以后修改和获得数值
    action类型用于绑定事件类型，把ui控件的事件和代码绑定在一起，用于控件的事件触发

if let语句，如果获得的数值不是nil，则把数值赋值给制定的代码，例如：
     if let var abc=UIController.input() {
         abc= //一定不是空值
     }
控件名称.resignFirstResponder 用于将控件失去焦点
控件名称.registerFirstResponder 用于重新获得控件的焦点
