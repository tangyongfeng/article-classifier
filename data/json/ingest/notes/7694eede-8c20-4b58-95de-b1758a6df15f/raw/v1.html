  • html 基础
    1. 批量注释 ctrl +/
    2. shift +alt 按住不放，鼠标向下拉，即可多行编辑相同的内容。
    3. 输入link:css可自动补全css整句
    4. 输入#box，则会补全一个id为box的div
    5. <p></p> 段落标签
    6. <br></br> 换行标签
    7. <em></em> 倾斜字体 <strong></strong>粗体字
    8. <div></div> 块分层
    9. <span></span>行分层
    10. <pre></pre>将文本转换为等宽字体， 忠实表达源码中的控股和换行。例如显示网页中的代码
    11. <blockquote></blockquote>块引用，左边右边都有缩进
    12. 制作表格 ：直接输入 table>tr*8>td*3{$}将会自动生成一个8x3的表格，其中用$填充123。ps：可以用{$$}试试看

    1. 链接外部样式表， 其中style.css为样式文件

    1. 其中rel="stylesheet"必不可少，表示页面引入一个样式表。
    <link rel="stylesheet" href="style.css">
    <style> </style>
    1. 链接外部样式可以通过 .p1 的 class方式调用，也可以通过 #p2 的id方式调用，效果和性能一样。
    2. 并列选择，（html,body等）,使用逗号间隔，可以直接使用html的主要标签来定义css样式，定义之后直接再对应的标签上生效。
    3. 后代（子项）选择器，使用空格间隔 ，这种方式可以跳过多重间隔生效，例如body中包含em ，则可以使用body em  { color:read}来定义包裹再body中的em 的样式。
    4. 父子选择器，使用>间隔，这种方式只能在标签的父子关系时生效。例如 p>strong
    5. 相邻选择器，使用+间隔，这种方式只能在两个相邻的标签中发挥作用。例如 p+h1
    6. 兄弟选择器，使用~间隔，这种方式将会在间隔之后的每一个相同的标签发挥作用，例如h1~h2 ，实现的效果则是h1h2h2h2h2h2h2h2h2h2h2 只要相邻的相同标签，都会有效果。
    7. 属性选择器，html中， <h1 title="demo">什么<h1>中的demo 就是这个标签的属性，可以直接通过属性的分类来定义样式.使用[]标记 。例如：h1[title]{color:red;} 。进阶，还可以根据属性的值来设定样式，使用方法如: hi[tile="demo"] {color:red;}
    8. 动态伪类选择器
a:link{
/* 有链接文字 */
    text-decoration:none; 
    /* 下划线去掉 */

    color: aqua;

    /* 有链接时,鼠标在文字上悬停时 */

}

      • [class *="col"] 通配有col元素的字符的类（无论前后）。
      • [class ^="col"] 通配col开头后续字符的类
      • [class $="col"] 通配col之前有文字的类
  • css长度单位
    • 写成div.size表示应用了size类的div。
    1. 像素px ：就是像素了
    2. em,指定文字是父级或者默认大小的倍数。1em 一倍。 2em两倍
    3. rem 相对根元素（html）的大小
    4. vw 是相对与窗口宽度的百分单位.和高度无关，只和宽度有关。
    5. vh是相对与窗口高度的百分单位
    6. vmax 是指文字的大小宽度和高度相比较大者有关。调整浏览器的宽度或者高度，改文字的大小也会随着浏览器的大小而变化。
    7. vmin是指问的大小宽度和高度相比较小者有关，调整浏览器的宽度或者高度，文字的大小也会随着浏览器的大小变化。
    8. %之前的标准定义%是一个长度单位，但是现在标准已经被修订为%是一个数据类型。100%=1em
  • css颜色的设置
    1. 使用颜色名称设置颜色
    2. 使用十六进制设置颜色，前面需要加#
    3. 使用rgb(0~255,0~255,0~255） 来设置颜色，同时可使用rgba(0~255,0~255,0~255,0~1)来设置，最后一位为透明度，小数。

      • section：主要用于一段主题性内，一般包括一个或多个标题。
      • article： 主要描述一段具体的、可以脱离上下文的内容。
    • nav，aside区别
      • nav元素是一个可以用作页面导航的链接组，其中的导航元素链接到其他页面或当前页面的其他部分。并不是所有的链接组都要被放进nav元素，只需要将主要的、基本的链接组放进nav元素即可。
        • 常见应用场景：
          • 导航
          • 页内导航
          • 翻页
      • aside元素用来表示当前页面或文章的附属信息部分，它可以包含与当前页面或主要内容相关的引用、侧边栏、广告、导航条，以及其他类似的有区别于主要内容的部分

  • 定位
    • position

absolute 生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。 元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
fixed 生成固定定位的元素，相对于浏览器窗口进行定位。元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。
relative 生成相对定位的元素，相对于其正常位置进行定位。因此，"left:20" 会向元素的 LEFT 位置添加 20 像素。
static 默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
sticky 粘性定位，该定位基于用户滚动的位置。它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。
          注意: Internet Explorer, Edge 15 及更早 IE 版本不支持 sticky 定位。 Safari 需要使用 -webkit- prefix (查看以下实例)。
inherit 规定应该从父元素继承 position 属性的值。
initial 设置该属性为默认值，详情查看 CSS initial 关键字。
